Keep Calm, Write On

Keep Calm and Write On is an interactive speed writing game, where two players take turns to create a story with only 30 seconds on each turn to write a paragraph to build a story.
Technologies used: Javascript, jQuery, Bootstrap, HTML, CSS Play here.

MusicBucket

MusicBucket was created by four people, allowing users to categorize their favorite Spotify tracks by moods and organizing these moods in playlists (buckets) built in the application. Users are required to be authenticated by Facebook in order to create their personalized playlists. Technologies used: Ruby, Rails, Bcrypt, Spotify API, AJAX, PostgresSQL, Javascript, jQuery, Bootstrap, HTML, CSS Play here.

Open Mic

OpenMic is a Node.js chat application where users can upload audio in real-time, using the SoundCloud API. This allows for easy collaboration on music projects between musicians. Technologies used: Node.js, Socket.io, Redis, SoundCloud API, Javascript, jQuery, HTML5/CSS3, Bootstrap
